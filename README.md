# TrendPulse Data Pipeline

This project collects, processes, analyzes, and visualizes trending stories from Hacker News.

## Tasks
- Task 1: Data Collection (API → JSON)
- Task 2: Data Cleaning (JSON → CSV)
- Task 3: Data Analysis (Pandas + NumPy)
- Task 4: Visualization (Matplotlib charts)

## Output
- Clean dataset: data/trends_analysed.csv
- Charts: outputs/

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
